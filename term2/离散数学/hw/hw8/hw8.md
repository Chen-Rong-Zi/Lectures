## Problem 3
设 A,B 为可数集，证明：
(1) A ∪ B 是可数集；
```
设|A| = m, |B| = n
所以A ∪ B <= m + n
所以A ∪ B 是可数集；
```

(2) A × B 是可数集.
```
设|A| = m, |B| = n
所以A × B = m * n
所以A × B 是可数集；
```

## Problem 10
证明：可数多个可数集的并集是可数的。
```
将多个可数集并排排列，其中每个元素一一对应,
1, 2, 3, 4,  5,  6...
2, 3, 4, 5,  6,  7...
4, 6, 8, 10, 12, 14...
我们可以由上到下地去数
1, 2, 4   , 2, 3, 6,  4, 5, 10,  ...
所以可数多个可数集的并集是可数的。
```
