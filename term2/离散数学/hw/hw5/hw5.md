## Problem 4
如果集合 A、B、C 满足下述条件，你能判断 A = B 吗，请说明理由或者给出例子？

$$ a. A ∪ C = B ∪ C $$
```
答：令A = {1}, B = {2}，C = {1, 2}
上式成立，但A != B
```
$$ b. A ∩ C = B ∩ C $$
```
答：令A = {1, 2, 3}, B = {2, 3, 4}, C = {2}
上式成立，但A != B
```
$$ c. A ∪ C = B ∪ C 并且 A ∩ C = B ∩ C $$
```
答：
假设A != B
不妨设 Z: ∃x, x ∈ A ∧ x ∉ B

1. 若 Y: x ∉ C
则 P: x ∈ A ∪ C
则 Q: x ∈ B ∪ C
Y ∧ P ∧ Q → M: x ∈ B
M 与 假设 Z 矛盾, Z不成立

2. 若 Y: x ∈ C
则 P: x ∈ B ∩ C
Y ∧ P -> Q: x ∈ B
Q与Z矛盾， Z不成立

综上A == B
```


## Problem 5
$$ 令 A 和 B 为全集 U 的子集。证明 A ⊆ B 当且仅当 \overline{B} ⊆ \overline{A}，这里 A 指 U − A。 $$

$$∀x, x ∈ \overline{B}$$
$$因为 A ⊆ B → x ∉ A ∧ x ∈ \overline{A}$$
$$由于 x 是 B 的补集中的任意元素，且x ∈ \overline{A}$$
$$所以 \overline{B} ⊆ \overline{A}$$


## Problem 6
证明如果集合 A，集合 B 是全集 U 的子集，则
$$ a. A ⊕ A = ∅ $$
$$   A ⊕  A$$
$$= A ∪ A - A ∩ A$$
$$= A - A$$
$$= ∅$$

$$ b. A ⊕ U = \overline{A} $$
$$    A ⊕ U$$
$$= A ∪ U - A ∩ U$$
$$= U - A$$
$$= \overline{A}$$

$$ c. A ⊕ B = B ⊕ A $$
$$左式：$$
$$= A ∪ B - A ∩ B$$
$$= B ∪ A - B ∩ A$$
$$= B ⊕  A$$

$$ d. (A ⊕ B) ⊕ B = A $$
$$= A ⊕  (B ⊕  B)$$
$$= A ⊕ ∅$$
$$= A$$
