# Problem 2
证明A × B != B × A 除非 A = B, 其中A和B均为非空集合
```
假设(A × B = B × A) ∧ (A != B)
不妨设 x ∈ B ∧ x ∉ A
所以 ∃ y ∈ A, (x, y) ∈ B × A
所以 (x, y) ∈ A × B
所以 x ∈ A
与假设矛盾，故A × B != B × A 除非 A = B
```
