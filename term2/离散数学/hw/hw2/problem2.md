## Problem 3
证明 (p → q) → r 和 p → (q → r) 不是逻辑等价。
```txt
  (p → q) → r
≡ ¬(¬p ∨ q) ∨ r
≡ (p ∧ ¬q) ∨ r
≡ (r ∨ p) ∧ (r ∨ ¬q)

  p → (q → r)
≡ ¬p ∨ (¬q ∨ r)
≡ ¬p ∨ ¬q ∨ r

所以二者不是逻辑等价
```


## Problem 4
证明 (p → q) → (r → s) 和 (p → r) → (q → s) 不是逻辑等价。
```txt
若r ≡ T, s ≡ ⊥
左式
  (p → q) → (r → s)
≡ (p → q) → ⊥
≡ p → q

右式
  (p → r) → (q → s)
≡ (p → T) → (q → ⊥)
≡ T → q
≡ q
此时二者不是逻辑等价
所以题中两式不是逻辑等价
```
